How to Build Derby For Splice Machine.

1.  Build the code.
	ant clean buildsource testing buildjars
4.  Go into maven2 directory.
	cd maven2
5.  Install Locally to your own Maven Repo
	./SpliceBuild.sh 'gpg passphrase'
6.  Push to nexus
	./SpliceDeploy.sh 'gpg passphrase'
